# 🏨 Hotel Booking System

A full-stack hotel booking system inspired by Booking.com

## Tech Stack
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Auth:** JWT
- **Payments:** Stripe
- **Email:** Nodemailer

## User Roles
- Guest — search & book hotels
- Hotel Owner — list & manage properties
- Admin — manage the platform

## Getting Started

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm start
